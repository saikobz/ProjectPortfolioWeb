# Glossary

## Portfolio (this site)

A bilingual landing page for job applications. Primary goal: a recruiter reads it in under 60 seconds and clicks through to GitHub, CV, or a live project demo. The site itself is not a Full Stack demo; backend skills are shown via separate project case studies.

## Audience

Recruiters and hiring managers evaluating a junior Full Stack candidate.

## Candidate profile

Junior level. Comfortable with React, Vue, Next.js, TypeScript, Node.js, REST APIs, Python, MySQL, and Tailwind CSS.

## Content scope

Two Full Stack project case studies at launch. Bilingual (Thai + English). Default locale: Thai. Language switcher toggles content on the same URL (no `/th` or `/en` route split). Neither project has a live demo at launch; GitHub and on-site media (screenshots, diagrams) carry the proof.

## HarmoniQ

AI music stem separation and mastering. Stack: Next.js, React, TypeScript, Tailwind CSS, Python, FastAPI, PyTorch. Public repo: `saikobz/project-music`.

## Contact

Direct links only: email, LinkedIn, and GitHub. No contact form.

## CV

Downloadable PDF from the site (e.g. `public/cv.pdf`). Primary CTA in Hero and repeated in Contact section.

## DocuMind

Team knowledge hub with document search and AI summaries. Display name for the former "AiProject" / knowledge hub concept. Stack: Next.js, React, TypeScript, Tailwind CSS, Node.js, Supabase, PostgreSQL. Public repo: `saikobz/AiProject`. Flagship case study on the portfolio site.

## Visual design

Light, minimal theme: white/gray surfaces, strong typography, single accent color. No dark/light toggle in v1. Bilingual UI with Thai as the default locale.
